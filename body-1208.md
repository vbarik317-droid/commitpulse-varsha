## Description — What does this PR do? Which issue does it fix?

This PR implements the requested Versus Mode feature for "Multiplayer Monoliths". Users can now append the `?versus=username2` parameter to the streak API route to compare their commit activity with a friend side-by-side on the same SVG canvas. The `generateVersusSVG` generator offsets their X translations so both isometric cities are rendered beautifully side-by-side with a divider and a "VS" badge.

Fixes #982
